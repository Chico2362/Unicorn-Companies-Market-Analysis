# Unicorn-Companies-Market-Analysis
This project explores global unicorn companies (valued over $1B) using SQL, SQLite, and Python to analyze industry trends, valuation patterns, and geographic distribution between 2019–2021.
# Project Overview
Designed a normalized relational database (4 tables: companies, industries, funding, dates)
Loaded 100 global unicorn companies into SQLite
Performed advanced SQL analysis using:
CTEs
Multi-table joins
Aggregations
STRFTIME date filtering
Conducted industry trend and yearly growth analysis (2019–2021)
Built visualizations using Matplotlib and Seaborn
Generated business insights on valuation trends and market concentration
# Key Findings
Fintech was the leading sector (22 new unicorns, avg valuation $7.86B)
2021 showed the highest unicorn growth (34 new companies)
North America and Asia dominate both in count and total valuation
Top valued companies include Bytedance ($140B), SpaceX ($127B), and SHEIN ($100B)
